# Referência de Ferramentas e Logos

Este documento lista todas as ferramentas externas utilizadas pelo framework, com links oficiais e referências aos seus logos.

## Ferramentas de Rede

### Nmap
- **Descrição**: Scanner de rede e auditoria de segurança
- **Website**: https://nmap.org/
- **Logo**: `assets/logos/nmap.png`
- **Licença**: GPL
- **Uso no Framework**: Port scanning, service detection, OS fingerprinting

### Aircrack-ng
- **Descrição**: Suite completa para auditoria de redes Wi-Fi
- **Website**: https://www.aircrack-ng.org/
- **Logo**: `assets/logos/aircrack.png`
- **Licença**: GPL
- **Uso no Framework**: WiFi scanning, packet capture, WEP/WPA cracking

### Tcpdump
- **Descrição**: Analisador de pacotes de rede
- **Website**: https://www.tcpdump.org/
- **Logo**: `assets/logos/tcpdump.png`
- **Licença**: BSD
- **Uso no Framework**: ARP sniffing, packet analysis

## Ferramentas Web

### DirBuster / Gobuster
- **Descrição**: Ferramenta de enumeração de diretórios e arquivos web
- **Website**: https://github.com/OJ/gobuster
- **Logo**: `assets/logos/gobuster.png`
- **Licença**: Apache 2.0
- **Uso no Framework**: Web directory enumeration

### cURL
- **Descrição**: Ferramenta de linha de comando para transferência de dados
- **Website**: https://curl.se/
- **Logo**: `assets/logos/curl.png`
- **Licença**: MIT-like
- **Uso no Framework**: HTTP requests, header analysis

## Ferramentas OSINT

### Whois
- **Descrição**: Cliente para consultas de registro de domínios
- **Website**: https://www.whois.com/
- **Logo**: `assets/logos/whois.png`
- **Licença**: Varies
- **Uso no Framework**: Domain information gathering

### DNSUtils (dig, nslookup)
- **Descrição**: Ferramentas de consulta DNS
- **Website**: https://www.isc.org/bind/
- **Logo**: `assets/logos/bind.png`
- **Licença**: MPL 2.0
- **Uso no Framework**: DNS queries, domain resolution
