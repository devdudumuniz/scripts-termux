# Documentação do Projeto

Bem-vindo à documentação oficial do DM Pentest Framework. Esta seção contém guias detalhados, tutoriais e a documentação da API para ajudar usuários e desenvolvedores a tirar o máximo proveito do projeto.

## Seções

-   `guides/`: Guias passo a passo e tutoriais para usar as ferramentas e módulos do framework. Inclui casos de uso práticos e exemplos do mundo real.
-   `api/`: Documentação técnica da API interna do framework, destinada a desenvolvedores que desejam criar novos plugins ou integrar o framework com outras ferramentas.
-   `contributing/`: Guias detalhados sobre como contribuir para o projeto, incluindo padrões de código, fluxo de trabalho de pull requests e como reportar bugs.
